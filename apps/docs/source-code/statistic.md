## Statistics

### 堆叠

统计函数处理的数据不是我们的原始表格数据，而是一个和每个通道绑定的对象，参考下面的例子

```javascript
// 在 x = 0 的位置堆叠，y 为每一个种类占的值
const raw = [
  { x: 0, y: 1 },
  { x: 0, y: 2 },
  { x: 0, y: 3 },
];

// 和每个通道绑定的对象
const data = {
  index: [0, 1, 2], // 三个几何图形
  values: {
    x: [0, 0, 0],
    y: [1, 2, 3],
  },
};
```
